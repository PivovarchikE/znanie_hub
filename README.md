# Znanie Hub 🎓

**Znanie Hub** — это веб-платформа для управления учебным процессом, ориентированная на взаимодействие преподавателей и учеников.
Система позволяет структурировать учебные материалы, назначать и проверять домашние задания, а также отслеживать прогресс обучения.

Проект реализован как серверное Django-приложение с рендерингом HTML-страниц и поддержкой работы с файлами.

---

## 🛠 Технологический стек

* **Backend:** Python 3.12+, Django
* **Database:** PostgreSQL (через Docker)
* **Frontend:** HTML5, CSS, JavaScript (Django Templates)
* **File Storage:** локальное хранение (`media/`)
* **Package Manager:** Poetry
* **Testing:** Pytest

---

## 📂 Структура проекта

```text
.
├── config/             # Настройки проекта, ASGI/WSGI и маршрутизация
├── courses/            # Основная бизнес-логика обучения
│   ├── assignments/    # Логика обработки заданий
│   ├── models.py       # Section, Topic, Homework, HomeworkFile и др.
│   ├── services.py     # Бизнес-логика (сервисы)
│   ├── views.py        # Обработчики запросов
│   └── templates/      # Шаблоны страниц курсов и заданий
├── users/              # Пользователи и роли
│   ├── models.py       # User, роли, профили
│   ├── forms.py        # Формы регистрации и редактирования
│   └── templates/      # Шаблоны авторизации и профиля
├── other_pages/        # Дополнительные страницы проекта
├── static/             # CSS, JS, изображения
├── media/              # Загружаемые файлы (аватары, задания, ответы)
├── tests/              # Тесты (models, views)
├── docker-compose.yml  # Конфигурация PostgreSQL
├── pyproject.toml      # Зависимости проекта (Poetry)
└── manage.py           # Утилита управления Django
```

## ⚙️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/PivovarchikE/znanie_hub.git
cd znanie_hub
```

---

### 2. Установка зависимостей

```bash
poetry install
poetry shell
```

---

### 3. Запуск базы данных

```bash
docker-compose up -d
```

---

### 4. Применение миграций

```bash
python manage.py migrate
```

---

### 5. Создание суперпользователя

```bash
python manage.py createsuperuser
```

---

### 6. Запуск сервера

```bash
python manage.py runserver
```

Приложение будет доступно по адресу:

```
http://127.0.0.1:8000
```
